Baidu Lite 优化百度的视觉体验

发表于 2016-05-13   |   分类于 Work ， Code    |   次阅读   |  

百度流氓！

在中文搜索上，百度还是有可取之处的，然而百度的搜索体验实在不敢恭维，无所不在的推广、无用的按钮、首页上垃圾的板块、结果页糟糕的排版无一不在一次一次弓虽女干着我们的眼睛。

虽然百度的替代品很多，但是有些时候我们还是会需要用到它。

今天放出「百度 Lite」的测试版供大家试用，对百度的搜索界面进行了多处修改和优化。

★ 如何使用？

  1. 使用 Chrome 内核浏览器或 Firefox 浏览器；

  2. 下载安装 Stylish 扩展程序；

    Chrome 应用商店（或者通过 不需搭梯子的非官方商店 安装）
    Firefox 扩展程序
    QQ 浏览器插件市场
    猎豹浏览器应用市场

  3. 点击 一键安装 Baidu Lite。

切换日夜间模式也在该页面，可以从 Stylish 插件后台选项中前往。

有关推广链接

百度上下两侧的推广链接的 class 是随机生成的，我现在的能力不可以在 css 中将他们去除，可以选择 「屏蔽百度推广」 这个插件来完成屏蔽工作。

优化项目:
搜索首页

  • 修改了页面背景颜色；
  • 优化了百度的综艺体 Logo 和 #00c 的蓝色；
  • 优化了搜索框的样式；
  • 优化了搜索按钮的样式；
  • 优化了搜索建议的样式；
  • 优化了导航的样式；
  • 去除了无用的导航条；
  • 去除了页面 Footer 的二维码和版权信息；
  • 新增了搜索框动画（- -）。

结果页面

  • 增加了 Tab 的高度视觉上更轻松；
  • 更改了行高；
  • 优化了 Tab 的颜色；
  • 优化了超链接和强调文字的颜色；
  • 优化了下划线的显示效果；
  • 优化了特殊区域的 Tab 和 阴影效果；
  • 优化了页码按钮样式；
  • 去除了页面右侧的推荐。
